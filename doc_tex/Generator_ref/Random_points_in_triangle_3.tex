\ccDefinition

The class \ccRefName\ is an input iterator creating points uniformly
distributed in a triangle. The default \ccc{Creator} is
\ccc{Creator_uniform_3<Kernel_traits<P>::Kernel::RT,P>}.

\ccOperations

\ccCreationVariable{g}
\ccConstructor{Random_points_in_triangle_3(Point_3& p, Point_3& q, Point_3& r, Random& rnd =
  default_random);}{%
  $g$ is an input iterator creating points of type \ccc{Point_3} uniformly
  distributed in the triangle with the vertices $ p, q, r$,
  i.e.~$\forall pt = *g: pt = \alpha * p + \beta * q + \gamma * r$~, where 
~$ \alpha, \beta, \gamma \in [0, 1] $~ and ~$ \alpha + \beta + \gamma = 1 $~.
  Two random numbers are needed from \ccc{rnd} for each point.
}
