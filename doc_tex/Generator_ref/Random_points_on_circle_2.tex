\begin{ccRefClass}{Random_points_on_circle_2<Point_2, Creator>}
\ccDefinition

The class \ccRefName\ is an input iterator creating points uniformly
distributed on a circle.  The default \ccc{Creator} is
\ccc{Creator_uniform_2<Kernel_traits<P>::Kernel::RT,P>}.
The generated points are computed using floating point arithmetic,
whatever the Kernel is, thus they are on the circle/sphere only up to
rounding errors.
