\begin{ccRefClass}{Random_points_in_surface_mesh_3<Point_3, C2t3, Creator>}

\ccDefinition

The class \ccRefName\ is an input iterator creating points uniformly
distributed inside a 3D mesh, taking into account the weight of each cell.
The default \ccc{Creator} is \ccc{Creator_uniform_3<Kernel_traits<P>::Kernel::RT,P>}.

\ccCreationVariable{g}
\ccConstructor{Random_points_in_mesh_3(C3t3& c3t3, Random& rnd = default_random);}{%
  $g$ is an input iterator creating points of type \ccc{Point_3} uniformly
  distributed (also taking into account the weight of each cell) inside a 3D
  mesh with the MeshComplex_3InTriangulation_3 $ c3t3 $,
  i.e.\ ~$ \forall pt = *g: pt = \alpha * p + \beta * q + \gamma * r + \delta * s$~,
  where ~$ \alpha, \beta, \gamma, \delta \in [0, 1] $~ and
  ~$ \alpha + \beta + \gamma + \delta = 1 $~, with \f$ p, q, r, s $~ being the
  vertices of a cell in c3t3.
  Four random numbers in total are needed from \ccc{rnd} for each point. More
  precisely, one random number is generated for picking a cell and then
  Random_points_in_tetrahedron_3::generate_points() is called, which in turn
  needs three numbers from `rnd` for each point.
}
