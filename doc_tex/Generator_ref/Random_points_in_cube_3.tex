\ccDefinition

The class \ccRefName\ is an input iterator creating points uniformly
distributed in a half-open cube.  The default \ccc{Creator} is
\ccc{Creator_uniform_3<Kernel_traits<P>::Kernel::RT,P>}.

\ccCreationVariable{g}
\ccConstructor{Random_points_in_cube_3( double a, Random& rnd =
  default_random);}{%
  $g$ is an input iterator creating points of type \ccc{Point_3} uniformly
  distributed in the half-open cube with side length $2 a$, centered
  at the origin, i.e.~$\forall p = \ccc{*g}:  -a \le p.x(),p.y(),p.z() < a$~. 
  Three random numbers are needed from \ccc{rnd} for each point.
}
